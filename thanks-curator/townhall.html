03  레슨런
Lesson 01

만드는 시간보다 판단하는 시간이 늘었다

"영상을 폴더에 넣고 Claude에 요청하면 설명문과 타임스탬프가 나온다.
내 일은 이제 쓰는 것이 아니라 고르는 것이 됐다."

영상 업로드 → Claude Code 분석 요청만으로 제목·설명문·타임스탬프 완성.
반복 작업이 사라지고, 콘텐츠 방향과 품질 판단에 집중할 수 있게 됐다.

Lesson 02

데이터는 있었다 — 분석할 시간이 없었을 뿐

"YouTube Studio 스크린샷을 공유하면 Claude가 수치를 읽고 보고서를 만든다.
반나절 걸리던 보고서가 30분 안에 끝난다."

데이터는 항상 있었지만 정리·분석·시각화까지 직접 하려면 시간이 너무 걸렸다.
스크린샷 하나로 보고서가 완성되면서 실행 속도가 달라졌다.
